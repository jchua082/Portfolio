BusMeet

MOBILE DEVELOPMENT, UX / UI DESIGN

Year:

2021

Skills:

React Native, JavaScript, UX Design, Figma

Project Scope

To create an integrated application where users can check bus information while interacting with other users, such as messaging, location sharing, and sharing their estimated time of arrival (ETA) seamlessly.

Introduction

As users of the SG BusLeh App, our team found it tedious to navigate between the bus app and updating your current location and ETA when meeting up with someone.

Thus, this served as an inspiration for our mobile application, BusMeet, to create an integrated app with all the necessary functions to meet up when travelling via public transport.

Designing Mockups

Our team utilised Figma to create prototypes of the BusMeet App. We decided to create assets with the school theme in mind.

We also created avatar designs where users can customise their profile image, making this app more interactive.

App Development

We utilised the Expo framework using React Native for our application development and Amazon Web Services (AWS) Amplify to set up the backend infrastructure of our application.
